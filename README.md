# Machine Learning with Python

Dieses Repository dokumentiert mein strukturiertes Lernen von **Python-Grundlagen und Machine-Learning-Methoden** im Rahmen des Moduls *Business Analytics / Machine Learning with Python* (FHNW).

Der Fokus liegt auf **Verständnis, sauberer Umsetzung und praktischer Anwendung** zentraler ML-Konzepte – nicht auf Black-Box-Nutzung von Frameworks.

---

## Inhalte & Lernziele

Das Repository deckt folgende Themen ab:

- Python-Grundlagen für Data Science  
  (Datentypen, Kontrollstrukturen, Funktionen, Listen, Dictionaries)
- Datenaufbereitung und -analyse  
  (NumPy, pandas, Datenbereinigung, Feature-Auswahl)
- Überwachtes Lernen  
  - k-Nearest Neighbors (k-NN)  
  - Lineare & logistische Regression  
  - Entscheidungsbäume
- Unüberwachtes Lernen  
  - Clustering (z. B. k-Means, Hierarchical Clustering)
- Modellbewertung  
  (Train/Test-Split, Accuracy, Precision/Recall, Confusion Matrix)
- Grundlegende ML-Konzepte  
  (Overfitting, Bias–Variance-Tradeoff, Feature Scaling)
